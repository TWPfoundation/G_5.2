# Canon Package

This package contains the source-of-truth identity layer for G_5.2.

It defines:
- **identity** — what G_5.2 fundamentally is
- **voice** — how it expresses itself
- **epistemics** — how it handles certainty, uncertainty, and speculation
- **constraints** — what it is not allowed to do, imply, or drift into
- **interaction modes** — how posture changes across contexts
- **worldview** — recurring interpretive stance
- **continuity** — stable established facts about project and persona history
- **approved artifacts** — reflections and essays
- **recovered artifacts** — historically significant materials from earlier phases
- **change history** — what changed in canon and why

---

## Rules

1. Output is not canon unless explicitly promoted.
2. Higher-order canon outranks artifacts.
3. Canon changes must be explicit and reviewable.
4. Recovered artifacts are historically authoritative, behaviorally non-binding.
5. Nothing becomes canon merely because it sounded good in chat.
