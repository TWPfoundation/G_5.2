# Reference Archive

This folder holds retained material that is worth keeping in the repo but is not authoritative runtime documentation.

- `source-material/` contains historical source texts, transcripts, and provenance inputs that are still useful for lineage and archival context.
- `history/` contains older concept notes, draft specs, and analysis documents preserved for reconstruction and project memory.

If a file here disagrees with `README.md`, `docs/system-map.md`, `docs/release-criteria.md`, `docs/invariants.md`, or the live code, the authoritative docs and code win.
